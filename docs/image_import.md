# Image Import

## Overview
The first step in processing multi-angular UAV data is importing images into Agisoft Metashape.

## Steps
1. Open Metashape and create a new project.
2. Go to `Workflow > Add Photos`.
3. Select the folder containing the UAV images.
4. Ensure all images are loaded and previewed correctly.

## Notes
- Ensure images have consistent naming conventions.
- Check for missing EXIF metadata.
